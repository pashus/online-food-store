# Онлайн магазин продуктов питания

## Установка

Установите зависимости приложения командой:

```sh
npm install
```

## Режим разработки

Запустите приложение в режиме разработки командой:

```sh
npm run dev
```

## Режим продакшн

Соберите приложение для продакшна командой:

```sh
npm run build
```

## Организация кода

1. **api** - содержит конфиг axios инстанса
2. **assets** - статика: картинки, шрифты, другое
3. **components** - ui и не ui компоненты
4. **constants** - файл с константами
5. **features** - фичи, к примеру слайсы для глобального стора
6. **hooks** - все хуки в приложении
7. **lib** - папка от shadcn/ui
8. **pages** - страницы приложения
9. **services** - содержит объекты с функциями, выполняющими запросы на сервер
10. **store** - глобальный store
11. **types** - все типы в приложении

На данный момент:

- **App.tsx** - используется для организации роутинга
- **main.tsx** - точка входа, в этом файле все провайдеры

Стилизация:

- В проекте используется **tailwind**, стилизовать через него
- Стилизация уведомлений также может проходить через **tailwind** либо через переменные **toastify** в **:root** в **index.css**
